## 1.Eclipseでの設定

### 1-1.タブを半角スペースに変更

1. 画面上部の「ウィンドウ」から「設定」を選択する。
2. 設定画面から、「一般」→「エディター」→「AnyEdit ツール」を選択
3. 「タブスペースの変換」と「タブをスペースに」にチェックをし「適用」を選択する

### 1-2.タブと半角スペースを表示

1. 画面上部の「ウィンドウ」から「設定」を選択する。
2. 設定画面から、「一般」→「エディター」→「テキスト・エディター」を選択
3. 「空白文字を表示」にチェックを付け、必用に応じて「可視性の構成」を行う

## 2.Git関連
